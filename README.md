# mm_pic_spider
get pictures from https://www.m131.cc/xinggan/

主要参考https://github.com/3inchtime/mmjpg_spider

直接运行Spider_m131.py即可。

已添加多线程支持，创建最多8个线程的线程池下载图片，本人实测下载987套图片耗时约10小时。为避免访问太过频繁被服务器切断故止步于此。

987套图片体积约5GB
